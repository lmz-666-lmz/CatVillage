# Vue3猫咪社交与健康管理APP开发步骤

## 第一阶段：项目初始化与架构搭建

### 步骤1：项目创建与环境配置
- 使用Vite创建Vue3项目
- 安装必要依赖（Pinia、Vue Router、Axios等）
- 配置ESLint和Prettier
- 设置TypeScript（可选但推荐）

### 步骤2：目录结构搭建
```
src/
├── api/                    # API接口服务层
│   ├── petProfile.ts       # 宠物数字档案管理服务
│   ├── social.ts           # 交友广场服务
│   ├── message.ts          # 好友消息服务
│   ├── emotion.ts          # 喵喵情绪台服务
│   └── aiAssistant.ts      # AI养育助理服务
├── utils/
│   ├── request.ts          # 网络请求封装
│   ├── date.ts             # 日期格式化工具
│   ├── helpers.ts          # 通用辅助函数
│   └── validators.ts       # 表单验证工具
├── stores/                 # Pinia状态管理
│   ├── user.ts             # 用户状态
│   ├── cats.ts             # 猫咪档案状态
│   ├── currentCat.ts       # 当前选中猫咪状态
│   └── index.ts            # 统一导出store
├── types/                  # 类型定义
│   ├── index.ts            # 通用类型定义
│   ├── cat.ts              # 猫咪档案类型
│   ├── social.ts           # 社交相关类型
│   ├── message.ts          # 消息相关类型
│   ├── emotion.ts          # 情绪识别相关类型
│   └── aiAssistant.ts      # AI助理相关类型
├── components/             # 公共组件
│   ├── LoadingSpinner.vue  # 加载动画组件
│   ├── BaseButton.vue      # 基础按钮组件
│   ├── BaseInput.vue       # 基础输入框组件
│   ├── BaseCard.vue        # 通用卡片组件
│   ├── CatSelector.vue     # 猫咪选择器组件
│   └── index.ts            # 统一导出组件
├── composables/            # 业务逻辑Composables
│   ├── useCatManagement.ts # 猫咪档案管理逻辑
│   ├── useSocialFeatures.ts # 社交功能逻辑
│   ├── useMessaging.ts     # 消息功能逻辑
│   ├── useEmotionAnalysis.ts # 情绪分析逻辑
│   ├── useAIAssistant.ts   # AI助理逻辑
│   └── index.ts            # 统一导出composables
├── mock/                   # Mock数据服务
│   ├── mockCats.ts         # 猫咪档案Mock数据
│   ├── mockSocial.ts       # 社交功能Mock数据
│   ├── mockMessages.ts     # 消息功能Mock数据
│   ├── mockEmotion.ts      # 情绪分析Mock数据
│   ├── mockAI.ts           # AI助理Mock数据
│   ├── mockHandler.ts      # Mock数据拦截器
│   └── index.ts            # 统一导出mock
├── views/                  # 页面视图
├── assets/                 # 静态资源
├── constants/              # 常量定义
│   ├── index.ts            # 应用常量定义
│   ├── apiEndpoints.ts     # API端点常量定义
│   └── catBreeds.ts        # 猫咪品种常量定义
├── router/                 # 路由配置
└── shims-vue.d.ts          # Vue文件类型声明
```

### 步骤3：网络请求封装
- 创建统一的axios实例
- 添加请求拦截器（自动添加Authorization头部）
